# Setup Instructions
## Clone the repository:
```bash
git clone https://github.com/sainarsareddy/KERNNautomationsassignment.git
```
## Navigate to the project directory:

```bash
cd flight-booking-app
```

## Install the dependencies:

```bash
npm install
```
## Start the development server:

```bash
npm start
```
